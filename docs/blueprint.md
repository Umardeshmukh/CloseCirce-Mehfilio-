# **App Name**: CircleShare

## Core Features:

- Circle Creation: Users can create private circles to invite close friends.
- Invitation System: Invite friends to join circles via unique, secure invite links.
- Content Posting: Share photos, videos, and text updates within a circle.
- Chronological Feed: Display posts within each circle in a chronological feed, free of algorithms.
- Post Interactions: Members can like and comment on posts within circles.
- User Profiles: Basic user profiles to display name and a profile picture. Let users generate an AI-driven profile blurb if they want; consider this profile blurb when mentioning that user in Circle updates.
- Settings Page: Enable users to edit their profile and manage circle settings, including a dark mode option.

## Style Guidelines:

- Primary color: Dark teal (#004040) for a sense of calm and trustworthiness in a dark theme.
- Background color: Dark gray (#222222), providing a comfortable contrast for a dark theme.
- Accent color: Blue (#4080FF) for interactive elements, standing out against the dark background to ensure visibility and engagement.
- Font: 'Inter' for a modern, clean aesthetic; use this sans-serif for both headings and body text.
- Minimalist icons to maintain a clean and simple UI, designed with high contrast for visibility in a dark theme.
- Clean, responsive layout inspired by BeReal and Instagram's simplicity, optimized for readability and usability in a dark theme.
- Subtle transitions for a smooth user experience, ensuring animations are not distracting in a dark theme environment. Use Framer Motion for smooth animation
